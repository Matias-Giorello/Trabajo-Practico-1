El objetivo de estre proyecto es poder demostrar lo aprendido a lo largo del cuatrimestre en el curso de programacion web fullstack de la UTN, para demostrarlo el profesor nos asigno la tarea de replicar una pagina web.
La pagina web fue replicada a partir de una captura donde tuve que usar las herramientas dadas en las clases para que quede lo mas fiel y parecido posible.
Durante el proyecto use el legunaje de etiquetado HTML, el lenguaje de estilos CSS y como herramientas aparte use google y diferentes IAs para poder ir aprendiendo sobre nuevas etiquetas.
Para poder organizar el HTML del proyecto decidi dividir la pagina en 7 secciones que fui editanto una a una mas adelante en CSS, las secciones  son: header, h1, textareaSection, inputSection, summarySection (decidi ese nombre porque es como un resumen breve del textarea), letterDensitySection y footer.
Lo que mas me costo fue la etiqueta de meter porque funcionaba diferente en cada navegador web entonces no podia hacer que me quede bien, tuve que recurrir a la ia para ayuda y que me explique pero ni ella pudo resolverlo asi que opte por dejarlo con fondo blanco.
